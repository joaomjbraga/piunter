# Instalação

## Via npm

```bash
npm install -g @bforgeio/piunter
```

## Via npx

```bash
npx piunter
```

## Do código fonte

```bash
git clone https://github.com/joaomjbraga/piunter.git
cd piunter
npm install
npm run build
npm link
```

## Requisitos

- Node.js 18+
- Linux (Debian, Ubuntu, Arch, Fedora, etc.)
- npm ou yarn para instalação via fonte
