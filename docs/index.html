piunter

CLI profissional para limpeza e otimização de sistemas Linux

Node.js 18+ TypeScript Linux MIT License

Instalação

npm install -g @bforgeio/piunter

Ou via npx:

npx @bforgeio/piunter

Uso Rápido

# Modo interativo
piunter

# Limpar tudo
piunter --all

# Analisar espaço
piunter --analyze

# Simular limpeza (dry-run)
piunter --all --dry-run

# Limpar módulos específicos
piunter --npm --cache --yarn

Todas as Opções

| Flag              | Descrição                     |
|-------------------|-------------------------------|
| --all            | Todos os módulos              |
| --cache          | Cache do usuário              |
| --npm            | Cache do NPM                 |
| --yarn           | Cache do Yarn                |
| --pnpm           | Cache do PNPM                |
| --flatpak        | Flatpak                      |
| --snap           | Snap                         |
| --docker         | Docker                       |
| --logs           | Logs do sistema              |
| --packages       | Gerenciador de pacotes       |
| --large-files    | Arquivos grandes             |
| --appimage       | AppImages                    |
| --thumbs         | Miniaturas                   |
| --recent         | Arquivos recentes            |
| --analyze        | Apenas analisar              |
| --dry-run        | Simular (sem executar)       |
| --force          | Pular confirmação            |

Recursos

  • Detecção automática de distribuição Linux
  • Suporte a APT, Pacman e DNF
  • Modo interativo com seleção por checkbox
  • Dry-run para simular antes de executar
  • Elevação automática de privilégios (sudo)
  • Sistema de plugins para extensibilidade
  • Shell completion (bash/zsh)
  • Config file personalizável
  • Testes unitários

Segurança

  • Nunca executa operações destrutivas sem confirmação (exceto --force)
  • Modo --dry-run para testar antes de aplicar
  • Detecção de comandos disponíveis
  • Tratamento robusto de erros
