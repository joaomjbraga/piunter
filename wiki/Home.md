# Home

Bem-vindo à documentação do piunter!

## O que é o piunter?

O **piunter** é uma CLI (Interface de Linha de Comando) profissional para limpeza e otimização de sistemas Linux.

## Características Principais

- Detecção automática de distribuição Linux
- Suporte a múltiplos gerenciadores de pacotes
- Módulos de limpeza modulares
- Modo interativo e via flags
- Dry-run para simulações seguras
- Elevação automática de privilégios
